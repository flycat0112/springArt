# converter、 formatter
用来将有关联关系但不同类型的数据之间互相转换。
比如：String格式的时间字符串和java对象中Date对象之间的转换，
还有日期不同输出格式的String的转换
"19940112" 转换成"1994-01-12"等。

有时候需要通过控制Locale
